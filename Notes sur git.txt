		Verifier quels fichiers on a supprimé, ajouté ou modifié depuis la dernière modification enregistrée (commit)
	git add * | git add .
		Ajouter les modifications (visibles avec git status) au prochain enregistrement (commit)
	git commit -m "monCommentaire"
		Enregistrer les modifications qui ont été ajoutées avec git add (celles en vert avec git status) avec un commentaire expliquant ce que vous avez fait
	git push origin [maBranche]
		ex: git push origin pascale
		Envoyer votre dernier enregistrement (commit) vers le serveur lointain, sur votre branche personnelle (pas accessible pour les autres)
